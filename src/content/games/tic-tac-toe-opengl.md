This project is part of my earlier path through C++, graphics, and game experiments. It recreates Tic-Tac-Toe using a small OpenGL-oriented stack rather than a game engine.

It is useful as historical evidence of sustained interest in game behavior and rendering. It should not be presented as proof of current graphics-programming or engine specialization.
